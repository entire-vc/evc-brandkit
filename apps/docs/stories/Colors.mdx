## Usage

### CSS Variables

```css
/* All colors are available as CSS variables */
.my-component {
  background: hsl(var(--primary));
  color: hsl(var(--primary-foreground));
  border: 1px solid hsl(var(--border));
}
```

### Tailwind Classes

```html
<!-- Use semantic color classes -->
<div class="bg-primary text-primary-foreground">
  Primary colored box
</div>

<div class="bg-secondary text-secondary-foreground">
  Secondary colored box
</div>

<div class="bg-destructive text-destructive-foreground">
  Error state
</div>
```

## Theme Switching

To switch themes, import the appropriate CSS file:

```css
/* Base theme */
@import '@evc/tokens/css/entire.css';

/* Or for Spark */
@import '@evc/tokens/css/spark.css';

/* Or for Playground */
@import '@evc/tokens/css/playground.css';

/* Or for Team Relay */
@import '@evc/tokens/css/team-relay.css';
```
