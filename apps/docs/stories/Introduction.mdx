# EVC Brand Kit

Welcome to the Entire VC Design System documentation.

## Overview

This design system provides a unified look and feel for all Entire VC products:

- **SaaS Applications** (React, Svelte)
- **Marketing Sites**
- **Browser Extensions**
- **Telegram Mini-Apps**
- **PWAs**

## Packages

| Package | Description |
|---------|-------------|
| `@evc/tokens` | Design tokens (colors, typography, spacing) |
| `@evc/ui` | React UI components |

## Quick Start

```bash
# Install packages
pnpm add @evc/tokens @evc/ui

# Configure Tailwind
const evcPreset = require('@evc/tokens/tailwind');

module.exports = {
  presets: [evcPreset],
  // ...
};

# Import theme CSS
@import '@evc/tokens/css/entire.css';
```

## Usage

```tsx
import { Button, Card, CardHeader, CardTitle } from '@evc/ui';

function MyComponent() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Hello EVC</CardTitle>
      </CardHeader>
      <Button>Get Started</Button>
    </Card>
  );
}
```
