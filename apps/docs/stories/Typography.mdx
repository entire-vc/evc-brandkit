# Typography

The EVC design system uses **Inter** as the primary font family for its clean, modern appearance and excellent readability.

## Font Family

```css
--font-sans: 'Inter', system-ui, -apple-system, sans-serif;
--font-mono: 'JetBrains Mono', 'Fira Code', monospace;
```
